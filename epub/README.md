# EPUB

## Dependencies

- [pandoc](https://pandoc.org/)
- [fd](https://github.com/sharkdp/fd)
- [sd](https://github.com/chmln/sd)

## Assembly Line

1. Setup `typ` files
1. Copy images and adjust `svg` fill and background
1. Convert `typ` to `epub`

## Generate EPUB

Navige to the`epub` directory and run `./epub-script`.
